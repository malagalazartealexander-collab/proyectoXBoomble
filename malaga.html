Resumen de lo que recuerdo de JAVA

Sintaxis de java

Elementos clave incluyen el uso de llaves {} para bloques de código, el punto y coma ; para finalizar instrucciones, y la necesidad de que el código resida en clases. Cada programa debe tener al menos un método main para ser ejecutable, que sirve como punto de entrada.

estructura repetitiva

Las estructuras repetitivas en Java (también llamadas bucles o ciclos) son for, while y do-while, que permiten ejecutar un bloque de código múltiples veces. Se utilizan para automatizar tareas repetitivas, controlar la ejecución basándose en una condición y evitar la repetición de código.

POO

POO en Java se refiere a la Programación Orientada a Objetos, un paradigma que utiliza "objetos" (instancias de "clases") para organizar el código en unidades que agrupan datos (atributos) y comportamientos (métodos). Java está diseñado para este modelo, y sus principales conceptos son la encapsulación, la herencia y el polimorfismo, que facilitan la creación de software modular, reutilizable y fácil de mantener.

herencia

La herencia en Java es un concepto de la programación orientada a objetos que permite que una clase (la hija o subclase) herede propiedades y métodos de otra clase existente (la padre o superclase). Esto promueve la reutilización de código, hace que la estructura del programa sea más clara y facilita el mantenimiento. Se implementa usando la palabra clave extends y la clase hija puede reutilizar lo heredado, agregar sus propios miembros o modificar los métodos heredados
